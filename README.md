# File_Comparator
This project implements a web-based application that compares two text files and analyzes their similarity.
Text File Similarity Analyzer

Description

This project compares two text files and calculates similarity based on common words.

Technologies Used

Java
Spring Boot
HTML
JavaScript
How to Run

Open project in IntelliJ
Run FilecompareApplication.java
Open browser: http://localhost:8080
Input Format

Upload two .txt files
Files should contain plain text
Output

Common words
Different words
Similarity percentage
Notes

Case sensitive
Large files may be slow
